Bar Chart Demo Page

This bar-chart.js library provides functions that can generate basic bar charts and stacked bar charts based on the data provided; which scales accordingly with the viewport size.

The bar chart generated are easily customizable with the built in helper functions that makes the process user-friendly and code free.

Below is a sample chart generated with the drawBarChart() function and the following variables:

  • labels = ["Hunter", "Preist", "Mage", "Warlock", "Paladin", "Warrior", "Rouge", "Druid", "Shaman"]
  • testdata = [54, 63, 50, 75, 81, 48, 53, 73, 67]
  • opt = {height: "60vh", width: "90vw"}

Below is a sample stacked bar chart generated with the drawStackedBarChart() function and the above and following variables:

  • theme = ["#05668D", "#028090", "#00A896", "#00A896", "#F0F3BD"]
  • nestedtestdata = [[25, 29], [30, 33], [32, 35, 8], [40, 41], [20, 20, 8], [23, 30], [45 ,28], [20, 25, 22]]

Instead of taking in an array of integers as the data, this function takes in a nested array to generate a stacked bar; based on the values and the total sum of each nested array.
